# Entrega EDA - Estadística Multidimensional:

📊 **Primera Entrega – Estadística Multidimensional**

Estimados estudiantes,

Para esta primera entrega deberán presentar el avance inicial de su proyecto, el cual tiene como objetivo validar la comprensión del dataset y el desarrollo adecuado del análisis exploratorio.

Cada grupo debe trabajar con un dataset propio y desarrollar los siguientes puntos:

🔹 **1. Dataset**

- Descripción del conjunto de datos (fuente, número de registros y variables)
- Tipo de variables (numéricas y categóricas)
- Justificación de por qué el dataset es adecuado para análisis multivariado

🔹 **2. Limpieza de datos**

- Identificación y tratamiento de valores nulos
- Eliminación de duplicados (si aplica)
- Corrección de tipos de datos

🔹 **3. Análisis univariado**

- Medidas descriptivas (media, mediana, desviación estándar)
- Visualizaciones (histogramas, boxplots)
- Interpretación de cada variable relevante

🔹 **4. Análisis bivariado**

- Relación entre variables (correlaciones, scatterplots)
- Matriz de correlación (si aplica)
- Interpretación de las relaciones encontradas

### **🔹 5. Primeros hallazgos**

- Identificación de patrones iniciales en los datos
- Variables más relevantes para el análisis

### **🔹 6. Análisis multivariado básico**

Deben incluir:

- Matriz de correlación
- Coordenadas paralelas

🔍 **Interpretar:**

- ¿Qué variables están más relacionadas?
- ¿Qué patrones globales se observan?
- ¿Existen grupos de observaciones similares?

### **🔹 7. Medidas descriptivas multivariantes**

- Medidas de localización (media vectorial)
- Dispersión (varianza/covarianza)
- Dependencia (correlación)

🔍 **Preguntas guía:**

- ¿Qué tan dispersos están los datos?
- ¿Qué variables varían más?
- ¿Cómo se relacionan entre sí?

### **🔹 8. Interpretación global (LA MÁS IMPORTANTE)**

- ¿Qué patrones encontraron?
- ¿Qué variables dominan el comportamiento?
- ¿Qué conclusiones pueden obtener?
- ¿Qué tipo de problema podría resolverse con estos datos?

📌 **Entregables:**

- Informe en PDF (bien estructurado)
- Código en R (ordenado y reproducible)

⚠️ **Importante:**

No se evaluará únicamente la generación de gráficas. Es fundamental la interpretación de los resultados. Gráficas sin análisis no tendrán valoración alta.

Este avance es fundamental, ya que sobre esta base se desarrollará el análisis multivariado en la siguiente fase.
